# Fluent ORM

Fluent ORM for your Comely apps.

## Supported Adapters

* MySQL 5.7+
* SQLite3

## Prerequisites

* PHP >= 7.1.x
* [comelyio/database](https://github.com/comelyio/database) 

## Installation

`composer require comelyio/fluent`
